# skrap

Easily scrap web pages by providing json recipes

## Getting Started
Install the module with: `npm install skrap`

```javascript
var skrap = require('skrap');
var recipePath = "./recipe.json";

skrap(recipePath, {param: 'value'}, function(data) {
  	console.log(data);
})
```
